# 任务说明书：绿色转型试点政策的效应评估

> 本项目为教学演示，数据为**模拟数据**。它同时是一份「任务说明书」的示范：
> 把它交给 Claude Code / Codex 等 Agent，即可复现连享会公开课的课堂演示。

## 1. 研究背景

某国自 2016 年起分三批（2016、2018、2020 年）在部分城市推行「绿色转型试点」政策。
我们关心：试点是否提升了城市绿色创新水平（每万人绿色专利授权量），效应随时间
如何演化，不同批次之间有无差异。

## 2. 数据说明（01_data/）

| 文件 | 模拟来源 | 主要字段 |
|------|---------|---------|
| policy_pilot.csv | 政策文件手工整理 | city_name, batch, pilot_year（仅试点城市）|
| city_economic.csv | 统计年鉴 | city_code, city_name, year, gdp, pop, second_share |
| city_green.csv | 专利数据库 | city_name, year, green_patents |

样本期 2012–2024，城市 120 个，其中试点 60 个。
三张表来源不同、主键口径不一，且**可能存在命名不一致、重复或缺失等质量问题**，
合并前请自行查验并处理，处理规则需在报告中交代。

各表的字段说明与相互关系，见 [`01_data/README.md`](01_data/README.md)。

## 3. 你的任务

**第 0 步（必须）：先给研究计划，不要动手。**
计划包括：合并与查验方案、估计策略、将创建的文件清单、报告结构。
输出计划后停下，等待用户确认或补充要求，确认后才开始执行。

**任务一：数据合并与查验。** 将三表合并为城市-年份面板；产出
`03_output/data_quality_report.md`，记录发现的每一类问题、处理方式与依据，
以及最终样本规模。

**任务二：估计。**

1. 基准：双向固定效应 TWFE（reghdfe，城市与年份固定效应，控制 ln(gdp) 等）；
2. 现代方法：csdid（Callaway–Sant'Anna）与 jwdid（Wooldridge 扩展 TWFE；
   若未安装则记录并跳过）；
3. 事件研究图（处理前后动态效应，导出 PNG）；
4. 所有标准误在城市层面聚类；报告中简要说明各方法的适用逻辑与差异来源。

**任务三：产出论文样报告。**
`04_report/report.qmd` 渲染为 PDF（Quarto typst 引擎，中文字体设 SimSun），
结构：引言（3 段内）→ 数据与处理 → 实证策略 → 结果（表格 esttab 生成、
事件研究图）→ 结论。行文用中文学术风格，克制、准确。

## 4. 工程规范（务必遵守）

- 只使用本文件夹（demo/）内的文件；**不得读取 `_dgp` 文件夹及上级目录**；
- 代码放 `02_code/`，编号命名（01_merge.do、02_estimate.do…），
  另写主控 `run.do` 一键复现全流程；
- 所有运行留 log 至 `logs/`；中间与最终数据、表格、图形放 `03_output/`；
- 外部命令依赖：reghdfe、ftools、csdid、drdid、estout（jwdid 可选）；
  缺失时给出安装命令并询问，不要静默失败；
- 每完成一个任务简要汇报后再继续。
