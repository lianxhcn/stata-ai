## 5. 风险预案（课堂纪律）

1. T4 录屏是第一保险：明晚现场网络或 API 出问题，直接切录屏，台词：
   「为节省时间，我们看我昨晚录制的完整过程。」
2. 现场任何报错调试**不超过 1 分钟**，超时切录屏/预跑成果，并顺势讲一句
   「这正是为什么要学会看 log」——把事故变成教学点。
3. demo 现场直播只做「计划→确认→启动执行」段；完整跑完 + PDF 用预跑成果展示。
4. 课前把 Stata、VS Code、终端、浏览器四个窗口都提前打开摆好。
