## 2. 协变量

- `lngdp`：2012 年基期 lngdp_i ~ N(10.5, 0.6²)；此后每年增长 0.07 + N(0, 0.02²)。
- `pop`（万人）：基期 ~ N(450, 150²)，下限 80；每年增长 0.5% + N(0, 0.3%²)。
- `second_share`（第二产业占比，%）：基期 ~ N(45, 8²)，每年递减 0.3。

## 3. 处理分配（带选择性）

按基期 lngdp + N(0, 0.3²) 的噪声排序，取前 60 名为试点城市（经济更强的城市更可能
入选）；其中排名 1–20 为第一批 g=2016，21–40 为第二批 g=2018，41–60 为第三批 g=2020。

## 4. 结果方程

对城市 i、年份 t：

```
y_it = 6 + a_i + l_t + 0.8*(lngdp_it - 10.5) + tau_it + e_it
```

- a_i ~ N(0, 1.5²)；**试点城市额外 +0.8**（水平选择，FE 类方法可吸收）；
- l_t = 0.25*(t - 2012) + u_t，u_t ~ N(0, 0.2²)，同年所有城市共用同一实现；
- e_it ~ N(0, 1)；
- 处理效应（动态且分批异质，这是 TWFE 出偏的关键设计）：
  t ≥ g 时 `tau_it = base_g * (1 + 0.3*(t - g))`，否则 0；
  base_2016 = 1.5，base_2018 = 1.0，base_2020 = 0.6；
- 最后 `y = max(y, 0)`，保留 1 位小数，命名 `green_patents`。

## 6. 真值输出 → `working/dgp_truth.md`（不得放入 demo/）

基于生成时的 tau_it（删行去重前的完整面板）计算并列表：

1. 整体 ATT：所有处理后城市-年单元 tau 的均值；
2. 分批 ATT：三批各自处理后单元均值；
3. 分事件期 ATT：e = t − g = 0,1,...,8 各期均值；
4. 附注一句：预期 TWFE 因异质动态效应而偏离，csdid / jwdid 应接近上述真值。
